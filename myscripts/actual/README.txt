This is a short README file for our scripts:
-----------------------------------------------------
1. monitoring.sh
	Позволяет выполнять мониторинг тукущего статуса WF. При запуске с параметром auto автоматически перезагружает юниты с локдаун ошибкой
	Работает, если находится в папках с шагами upgrade-posservers/till1/posclients
_______________________________________________________________________________________________________
2.progressbar.sh
	Показывает текущий процент готовности каждой кассы на текущем шаге. За эталон берется значение соответствующего шага из обновления till1
	Для использования положите в папку upgrade-posclients
	
_______________________________________________________________________________________________________
3.ping_all_tills.sh xx####
	Пингует кассы указанного магазина и выводит список. Если не указать магазин - будут пинговаться ВСЕ кассы из CMDB
	
_______________________________________________________________________________________________________
4.mqsc_files.sh ie0000s001 rusmik
	Скрипт для сервер-инсталла в новом магазине. На вход принимает номер сервера и вашу домашнюю директорию. 
	Копирует в домашнюю директорию mqsc файлы с сервера, добавляет данные в файл agent.properties на сервере и копирует на сервер файл moveclass
	Скрипт и movefiles.clacc лежат на зонмастерах в папке /root/scripts/mqscfiles/
	
_______________________________________________________________________________________________________
5.cert_import.sh ie0000s001 
	Копирует ском сертификаты на указанный сервер, регистрирует их и выводит данные для проверки.
	Для использования cert_import.sh и скрипт import_certs.ps1 должны находится в папке /root/scripts/certs_import/
	Сами pfx серты должны находится в /root/certs
6.SVSfixer.sh 
	Скрипт для катовера. Должен лежать в папке cutover_wf
	Проходит по всему заданному posclients.list, проверяет соединение с SVS и, если оно не верное - добавляет FW правило
	Выдает сообщение по завершению
