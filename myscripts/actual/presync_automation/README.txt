Оба файла должны лежать в корневой папке ВФ.
Требует заполненного файла sites.list и листов клиентов/серверов.
/prepare-presync.sh создает папку pre-syncs и в ней -  экземпляр ВФ для пресинка для каждого стора отдельно.
Каждый ВФ может быть запущен отдельно от другого
./presync_status.sh выводит статус по всем созданным ранее пресинкам
